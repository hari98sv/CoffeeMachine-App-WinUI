☕ Coffee Machine Application

A modern, event-driven coffee machine simulation application built with WinUI 3, featuring a clean architecture implementation with CQRS pattern and mediator implementation.

🎯 Key Features
🏗️ Architectural Patterns

* Clean Architecture with clear separation of concerns
* CQRS Pattern with separate command and query flows
* Custom Mediator Pattern without MediatR library dependencies
* Event-Driven Architecture with domain and integration events
* MVVM Pattern for WinUI 3 presentation layer

☕ Domain Features

* Multiple beverage types: Coffee, Latte, Espresso, Cappuccino
* Extensible design for adding new beverages (Tea, Milk, etc.)
* Inventory management with stock tracking
* Order processing with status tracking
* Recipe management with ingredient system

🎮 UI Features

* Modern WinUI 3 interface
* Real-time order status updates
* Beverage selection with customization options
* Responsive design with async operations
* Status messaging and error handling

🚀 Getting Started
Prerequisites

* .NET 8.0 SDK
* Windows 10/11
* Visual Studio 2022 with WinUI 3 support
